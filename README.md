# 🚗 Telegram Racer

A fast-paced car racing game with global leaderboards and Telegram integration.

## Features
- 🌍 Global persistent leaderboard
- 📱 Telegram profile sync
- 🏆 Live rankings
- 📊 Score archiving
- 👥 Player avatars
